## Project Overview

This project demonstrates an end-to-end Distributed Machine Learning pipeline on a banking marketing dataset using Apache Spark, Hadoop MapReduce, Apache Hive, and Spark Structured Streaming.

The objective of the project is to build a scalable analytics and machine learning workflow capable of handling large datasets through distributed computing technologies while performing customer subscription prediction and real-time analytics.

## Technologies Used

* Python
* Apache Spark
* PySpark MLlib
* Hadoop MapReduce
* Apache Hive
* Spark Structured Streaming
* HDFS

* NumPy

---

## Project Workflow

### 1. Exploratory Data Analysis (EDA)

* Data loading and preprocessing
* Missing value analysis
* Feature exploration and visualization
* Customer behavior analysis
* Target variable analysis

### 2. Data Parallelism using Apache Spark

* Partition analysis
* Repartitioning strategies
* Distributed DataFrame operations
* Parallel execution and workload distribution

### 3. Machine Learning with Spark MLlib

* Feature engineering
* Logistic Regression
* Decision Tree Classification
* Random Forest Classification
* Model evaluation using Accuracy and AUC
* Hyperparameter tuning using Cross Validation

### 4. Real-Time Analytics using Spark Structured Streaming

* Streaming data ingestion
* Real-time customer prediction
* SQL queries on streaming data
* Window-based aggregations
* Watermarking for delayed events

### 5. Hadoop MapReduce

* Python Mapper and Reducer implementation
* Hadoop Streaming execution
* Distributed data processing and aggregation

### 6. Apache Hive

* Hive table creation
* SQL-based analytics
* Data warehousing operations
* Query execution and reporting

---

## Model Performance

Multiple machine learning algorithms were evaluated and compared.

**Final Selected Model:** Random Forest Classifier

The model was further optimized using cross-validation and hyperparameter tuning to improve predictive performance.

## Key Outcomes

* Built a scalable distributed machine learning pipeline.
* Demonstrated parallel data processing using Apache Spark.
* Implemented machine learning models using Spark MLlib.
* Generated real-time predictions using Structured Streaming.
* Executed distributed analytics using Hadoop MapReduce.
* Performed data warehousing and querying using Apache Hive.
